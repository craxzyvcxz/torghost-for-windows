# TorGhost (windows edition made by terror)

A simple Windows GUI tool that routes your internet through Tor using a SOCKS proxy.

## Features

* One-click Tor toggle
* Automatic tor.exe launch
* Proxy enforcement
* DNS flush
* Clean disconnect

## Requirements

* `tor.exe` must be in the same folder as this program

## Usage

1. Run as Administrator (recommended)
2. Click **Turn On**
3. Wait for Tor to connect
4. Done

## Important

This is **NOT a full VPN**.
Some applications may bypass the proxy and use your normal internet connection.

## Troubleshooting

If you get an error saying `tor.exe` is missing:

* Make sure `tor.exe` is in the same folder
* Or search for `tor.exe` on your system and run it manually

## Notice

This is a Python script.
To run it as Administrator easily, you may want to convert it into a `.exe` file.

## Notes

* The program can run without Administrator privileges
* Running as Administrator reduces errors and improves reliability
